## 安装启动
---
* 安装
	* 安装sql server2016等版本，安装功能选择包括服务器和管理工具。
* 启动
	* 启动sql server 管理配置工具配置进行服务状态管理

## 管理配置
---
启动sql server management studio连接数据库，进行数据库、安全性、管理等方面的配置。

* 数据库：进行数据库、数据的增查改删等操作。
* 安全性：进行登录用户名管理、角色管理、加密等管理。
